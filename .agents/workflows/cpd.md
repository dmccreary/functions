// turbo-all

1. Run `git add .` to stage any currently modified files.
2. Run `git commit -m "[Generate a good message based on the recent changes]"` to commit the changes.
3. Run `git push` to push to the remote repository.
4. Run `mkdocs gh-deploy` to deploy the mkdocs site.
