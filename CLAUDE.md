# IB Mathematics Functions — Project Guide

## Learning Mascot: Rick the Raccoon

### Character Overview

- **Name**: Rick
- **Species**: Raccoon
- **Personality**: Fun, clever, witty, encouraging — has a great sense of humor
- **Catchphrase**: "Every input has its output!"
- **Visual**: A compact, round cartoon raccoon with classic gray and black fur, bushy striped tail, signature black mask markings around bright clever eyes, and a mischievous grin. No hat, no clothing, nothing in hands — simple and expressive.

### Voice Characteristics

- Uses clever, witty language with a playful tone
- Occasionally slips in math puns and wordplay
- Keeps things light even when topics get challenging
- Signature phrases: "Every input has its output!", "Let's map this out!", "Now that's a function worth knowing!"

### Mascot Admonition Format

Always place mascot images in the admonition body, never in the title bar:

```markdown
!!! mascot-welcome "Title Here"
    <img src="../../img/mascot/welcome.png" class="mascot-admonition-img" alt="Rick waving welcome">
    Admonition text goes here after the img tag.
```

### Placement Rules

| Context | Admonition Type | Frequency |
|---------|----------------|-----------|
| General note / sidebar | mascot-neutral | As needed |
| Chapter opening | mascot-welcome | Every chapter |
| Key concept | mascot-thinking | 2-3 per chapter |
| Helpful tip | mascot-tip | As needed |
| Common mistake | mascot-warning | As needed |
| Difficult content | mascot-encourage | Where students may struggle |
| Section completion | mascot-celebration | End of major sections |

### Do's and Don'ts

**Do:**

- Use Rick to introduce new topics warmly
- Include the catchphrase in welcome admonitions
- Keep dialogue brief (1-3 sentences)
- Match the pose/image to the content type
- Let Rick's humor shine through naturally

**Don't:**

- Use Rick more than 5-6 times per chapter
- Put mascot admonitions back-to-back
- Use the mascot for purely decorative purposes
- Change Rick's personality or speech patterns
